# Release Candidate Notes

This zip is intended as a serious handoff for the next phase of Repurly v2.

## What it is

- a modern Next.js/Postgres SaaS foundation
- multi-platform-aware data model and adapter architecture
- live-first direction for LinkedIn, X, and Facebook Pages
- scaffolded future-platform support for Instagram Business, Threads, YouTube, and TikTok
- operator docs and provider checklists

## What it is not

- a verified seven-platform production deployment
- a substitute for provider app review and live credential testing
- a guarantee that every media type is complete across every platform

## Best next step

Use this as the core v2 codebase, then finish provider integrations in order of business value and approval complexity.
