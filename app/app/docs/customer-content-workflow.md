# Customer content workflow

## What customers can do
Customers can now:
- create manual drafts
- generate starter drafts
- choose text, single-image or carousel format
- attach uploaded assets
- save drafts privately
- submit drafts for review
- schedule ready posts for publishing
- monitor scheduled and review queue states from the customer workspace

## Format rules
- text: no assets attached
- single_image: exactly one asset attached
- carousel: two or more assets attached

## Scheduling actions
- Save draft: keep the post private in the workspace only
- Submit for review: write the post into the review queue with pending approval
- Schedule for publishing: self-approve the post and write it into the publish schedule

## Guidance surface
Customers can use `/workspace/help` for plain-English explanations of:
- onboarding steps
- status meanings
- format choices
- schedule vs review behavior
