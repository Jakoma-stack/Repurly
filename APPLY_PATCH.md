# Apply this patch

Copy these files over the matching paths in your existing Repurly repo, then redeploy.

Recommended checks after applying:

1. Set the LinkedIn env vars using either the standard names or a suffixed set plus `LINKEDIN_ENV_KEY`.
2. Confirm your LinkedIn app has the company-page permissions you request.
3. Reconnect LinkedIn from **Channels**.
4. Verify a company page appears in the target list and is set as the workspace default.
