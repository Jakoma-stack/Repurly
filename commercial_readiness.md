# Commercial readiness checklist

## App access
- [ ] homepage loads
- [ ] sign-in works
- [ ] sign-up works
- [ ] dashboard loads after sign-in

## Core workflow
- [ ] create post works
- [ ] choose LinkedIn target works
- [ ] company page vs personal profile is clear in Channels
- [ ] submit for approval works
- [ ] schedule/publish works
- [ ] activity/job detail works
- [ ] AI draft generation works
- [ ] generated draft batch review works

## Team administration
- [ ] settings loads
- [ ] operator controls visible
- [ ] support snapshot visible
- [ ] invite team member works
- [ ] accept invite page works
- [ ] member removal works

## Commercial surfaces
- [ ] marketing site copy matches in-app positioning
- [ ] pricing matches across website, app, and billing flows
- [ ] reports page is visible in navigation
- [ ] notifications page is visible in navigation
- [ ] checkout route works
- [ ] billing portal fails gracefully when no customer exists

## Trust-breakers
- [ ] no placeholder auth/connect buttons in main flow
- [ ] no confusing dead pages
- [ ] no outdated pilot pricing on live surfaces
- [ ] no builder-facing copy on public pages
- [ ] no hidden URL-only pages for core features
