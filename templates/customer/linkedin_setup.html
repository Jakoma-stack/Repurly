How this works

1. Connect the workspace tokenRepurly opens LinkedIn OAuth, exchanges the authorization code for an access token, and stores that token in the workspace integration record.
2. Auto-fill the personal posting identityAfter OAuth, Repurly calls LinkedIn’s member profile endpoint and builds a person URN in the format urn:li:person:{id}.
3. Switch to company-page posting when neededFor organization posts, keep the connected token but replace the author URN with the company page organization URN once page-admin access is confirmed.
4. Keep setup fee as an optionIf clients should not do this themselves, choose the setup-fee path and keep the operational notes here.
