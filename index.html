The launch workflow

1. Compose

Create or reopen the same LinkedIn workflow item with the right target selected.

2. Approve

Route content for review before it enters the publish queue.

3. Schedule

Queue the post for a scheduled publish window with a visible operational state.

4. Recover

Use state-aware actions like Edit post, Retry now, and Requeue target when needed.
