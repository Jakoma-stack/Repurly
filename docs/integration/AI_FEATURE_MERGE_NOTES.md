# AI feature merge notes

This build keeps the newer Repurly interface and merges over the missing capabilities from the older AI-focused build.

## Added in this merge

- multi-brand workspace support in the current UI
- AI draft generation inside the composer
- engagement inbox workflow
- lightweight lead pipeline
- richer brand profile fields to guide AI draft quality

## What this is intentionally not

- not a full live social inbox sync yet
- not a full CRM
- not automatic publishing from AI
- not broader multi-network scope

## Operating stance preserved

- LinkedIn-first
- workflow-first
- commercially realistic
- optimized for agencies and B2B teams managing multiple brands

## Notes

- Engagement is manual-first capture today. That is deliberate so the feature is usable now without over-claiming live sync coverage.
- AI-generated content is created as draft posts only.
- Multiple brands are available inside one workspace so agencies and multi-brand teams can keep separate voice and CTA inputs.
